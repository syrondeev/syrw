مجموعه ویجت‌های حرفه‌ای و کاربردی برای Elementor

== Description ==

SYRW Elementor Widgets یک مجموعه از ویجت‌های حرفه‌ای و کاربردی برای افزونه Elementor است که با رعایت استانداردهای کدنویسی وردپرس توسعه یافته است.

= ویژگی‌ها =

* ساختار حرفه‌ای و مدولار
* کدنویسی استاندارد وردپرس
* عملکرد بهینه
* سازگار با آخرین نسخه Elementor
* پشتیبانی کامل از RTL

== Installation ==

1. فایل پلاگین را آپلود کنید به `/wp-content/plugins/syrw-elementor-widgets/`
2. پلاگین را از بخش 'Plugins' فعال کنید
3. مطمئن شوید که Elementor نصب و فعال است
